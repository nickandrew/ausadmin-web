The multiple vote report looks like this example: 


  Multiple vote group: MULTI-995725824
    Date: 2001-05-23 10:43:06
    From: "First Name Last Name" 
    IPs:  144.134.6.96 209.132.220.42

    Date: 2001-05-23 10:44:07
    From: "Barbara Last Name" 
    IPs:  144.134.6.96 209.132.220.41

    Date: 2001-05-23 10:46:46
    From: "First Name Last Name" 
    IPs:  144.134.6.96 209.132.220.43

    Date: 2001-05-23 10:50:27
    From: "Sally S." 
    IPs:  144.134.6.96 64.4.17.54

    Date: 2001-05-23 10:54:01
    From: "Eleanor Kirkwall" 
    IPs:  144.134.6.96 64.4.30.147

  • The identifier MULTI-995725824 is used to tie all messages together which are considered by ausadmin to come from the same source.
  • Each message appears underneath
  • Date is the date and time the message was received by ausadmin.
  • From is the complete From header line of the message
  • IPs is the set of IP addresses associated with the message which can include intermediate mail relays and originating addresses from webmail servers.

Ausadmin uses the above information, plus other information obtained from the message contents, to decide whether a group of messages appeared to come from the same person.
